# Logo Assets

Please add your logo file here:

## Required File

Copy your logo from:
```
/mnt/data/A_logo_design_for_"PRYNEX"_is_presented_in_three_v.png
```

To:
```
logo.png
```

## Optional Variants

For best results, also create these variants:

- `logo.svg` - Scalable vector version
- `logo-light.png` - Light theme variant (if needed)
- `logo-dark.png` - Dark theme variant
- `logo-icon.png` - Icon only (square, for favicons)

## Recommended Sizes

- **Header Logo**: 160x40px (or similar aspect ratio)
- **Footer Logo**: 120x30px
- **Favicon**: 32x32px, 16x16px
- **Social Share**: 1200x630px

## Usage

The logo is referenced in:
- `src/components/Navigation.tsx` (header)
- `src/components/Footer.tsx` (footer)
- `src/app/layout.tsx` (metadata)

Current path used in code: `/assets/logo.png`
